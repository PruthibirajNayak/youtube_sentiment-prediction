# youtube_sentiment-prediction
In this project, I developed a web application using FastAPI that fetches comments from YouTube videos, performs sentiment analysis on each comment using TextBlob, and categorizes the comments into Positive, Neutral, and Negative groups. The application integrates a Python backend to handle data processing and a user-friendly frontend to display the results in a visually clear and organized manner. By utilizing the Google API to retrieve YouTube data and combining it with Natural Language Processing (NLP) techniques for sentiment analysis, the system demonstrates the effective use of modern web technologies for real-time data processing and sentiment evaluation. This approach can be further expanded to support more advanced sentiment analysis applications and practical real-world use cases.
